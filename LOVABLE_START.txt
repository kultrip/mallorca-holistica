Project: Mallorca Holística (MVP)

Goal
Create a high-conversion, multilingual website that helps people in Mallorca find verified wellbeing professionals and events, with two search modes:

Classic search (therapy / name / location + map)
AI orientation search (natural language need → suggested therapies + professionals + related content)
Important rules

Visitors/users have NO accounts in MVP.
Professionals and admins exist (login can be minimal/placeholder); main public site is the priority.
Map provider: Mapbox (use a placeholder env var token).
Languages: ES, CA, EN, DE. Implement i18n with a language switcher (default ES).
Professional verification requires two docs: professional_id + national_id (DNI/NIE/passport). Show “verified” badges only when approved.
Use these files as the source of truth

API contract: openapi.yaml (all request/response shapes)
Mock data: mocks/db.json
AI orientation examples: mocks/orient_examples.json
